# Claude Code collaboration guide

Read and follow `AGENTS.md` as the shared project and coordination policy.

- Inspect `git status` before editing.
- Preserve all pre-existing modified and untracked files.
- Do not overwrite, revert, or reformat unrelated work from Codex or the user.
- Keep work limited to the files required by the request.
- If Codex may be editing the same file, ask the user to assign ownership.
- Prefer separate roles, branches, or Git worktrees for parallel implementation.
- Report changed files and verification results when finishing.
